# Read me!!!

## Garden Gathering
Garden Gathering is a web application that is a place for people to read and post messages on a garden themed page.
We are focused on keeping a calm and cozy vibe.

# Functionality
We strive to make this page as fun as possible while keeping it simple.
Everyone is able to post a message/thread via a button and form on the first page. This post will be shown as a flower in the garden (every flower is a different post) your own post should be easily visable with a glow/border around the flower. 
Press the different flowers to view thier messages. Inside the thread you will be able to read the (title, user, message and possible images). You will also be able to comment on the post and also comment och other peoples comments. To be able to comment you will need a username. Username is a features we are still thinking about how to implement (maybe log in before accessing the page or we could just have a little box on the main page where you can type your user. if no username when trying to comment a prompt should tell you that you need a username). You also can like/save posts. This is our main "saving feature" where you will be able to see all your liked posts later. We want to implement a little house on the main page in the garden where all your posts and liked posts will be saved. Press the house to enter the house (this will change the background) while in your cabin you'll see your posts in a list and also the liked posts. 
"this is the main functionality we want, more can obviously be added so please tell us about your ideas"

# Design
We want a cozy and calm place where you can chill and talk about stuff.
We choose to make it a pixel art garden, perfect for the vibe we are striving for.
Right now we are using images as the background. The background changes to night and the images will change to a darker version. 
Besides the pictures our design will be very straight forward with a color palette for primary and accent colors for both dark and light -mode. Fonts for titles and text.
